# 🚀 CodeArena – Full Stack Competitive Coding Platform

A **production-grade competitive programming platform** built with real-world system design principles — featuring contests, real-time battles, AI assistance, and scalable backend architecture.

---

## 🌟 Key Highlights

- ⚡ Real-time 1v1 Battles using WebSockets  
- 🧠 AI-powered doubt solving & editorial support  
- 🏆 Contest system with leaderboard & timer  
- 💻 Integrated code editor + execution pipeline  
- 📊 User analytics & progress tracking  
- 🛠️ Admin dashboard for content management  
- 🔐 Secure authentication & role-based access  

---

## 🏗️ Tech Stack

### Frontend
- React.js (Vite)
- Redux Toolkit
- Socket.io Client
- Custom Hooks Architecture
- Modular Feature-based Design

### Backend
- Node.js + Express
- MongoDB (Mongoose)
- Redis (caching + real-time sync)
- Socket.io (real-time engine)
- JWT Authentication

### System Design Concepts
- Scalable API design  
- Event-driven architecture (WebSockets)  
- Separation of concerns (MVC + services)  
- Optimized state management  

---

## 📂 Project Architecture

```
Project
├── backend
│   ├── controllers       # Business logic (contest, user, submissions, AI, etc.)
│   ├── models            # MongoDB schemas
│   ├── routes            # API endpoints
│   ├── middleware        # Auth & role handling
│   ├── config            # DB, Redis, WebSocket configs
│   ├── Socket.io         # Real-time battle logic
│   └── utils             # Validators & helpers
│
└── frontend
    ├── feature           # Feature-based modular architecture
    ├── Shared            # Reusable components & hooks
    ├── services          # API & socket layer
    ├── store             # Redux state management
    └── app               # Routing & app-level logic
```
